Using Variables in CSS

In Solace, we wanted to give user maximum flexibility with style customization. That's why we prepared Solace css to utilize Devlom Configurator variables. There's one key file where all the magic happens: 

solace/template/colours.html.twig

Take a look at html,body tag: 

html, body {
  color: {{ current_preset.main_text_color }};
  background-color: {{ current_preset.main_background }};
}

As you can see both text color and background-color will utilize main_text_color and main_background preset variables. In connection with Twig, you can create very sophisticated and automated layouts.

Twig is default templating engine used in phpBB 3.1. To learn more about Twig, please take a look at documentation.

That's it. You may also Purge the cache and you're done.
