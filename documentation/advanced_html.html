Modify HTML

If you're a developer and you want to try adjusting theme a little bit deeper. We prepared list of most important files where you can find Html and Twig code to modify forum structure:

  • base.html.twig - initial twig values, forum defaults (Twig array)
  • overall_header.html.twig - top structure of the forum
  • overall_footer.html.twig - bottom structure of the forum
  • navbar_top.html.twig - Top navigation bar
  • navbar_header.html - Top navigation bar content
  • navbar_bottom.html.twig - Bottom navigation bar
  • slideshow.html.twig - Slideshow
  • topbar.html.twig - Topbar including menu and logo
