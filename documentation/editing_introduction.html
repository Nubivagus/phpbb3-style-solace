Editing Content

To create forum content like: slideshows, menus and lists we used YAML language. Using yaml allows you to edit forum content without messing in the actual html code. Just remember about few simple rules:

  • Tabs are NOT allowed, use spaces ONLY.
  • You MUST indent your properties and lists with 1 or more spaces.
  • All keys/properties are case-sensitive. ("ThIs", is not the same as "thiS")

We recommend following text/code editors to use with our yaml files:

  1. Atom - OS X/Windows/Linux - editor developed by Github. It's free and open source. Ready to work with yaml files.
  2. Notepad++ - Windows - A free editor for Windows. Please take a look at this small tutorial regarding yaml files.
  3. Bluefish - OS X/Windows/Linux - A free, open source editor especially for Linux..

Editing yaml files is done locally on your computer. After editing, please upload saved file to corresponding location on your forum server and purge the cache.
