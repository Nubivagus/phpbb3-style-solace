Slideshow

Make sure that Devlom Configurator extension is installed in your phpBB. Otherwise changes made to slideshow.yaml won't be visible in your forum.

In Solace, we keep slideshow data in solace/content/slideshow.yaml file.

Before you start modifying slideshow.yaml file, please perpare images and upload them to: 

solace/theme/images/slides/

For best appearance, make sure all of them are in the same width and height.

Slideshow script will automatically adjust height to your image size. If you want to force slideshow to display with certain height, you can do it by specifying dedicated option inside Devlom Configurator "Slideshow" section (described in next chapters).

Now, open solace/content/slideshow.yaml.

In solace/content/slideshow.yaml you can specify image name, slideshow title and description. Where name, you have to provide file name of the image uploaded to solace/theme/images/slides/.

Add as many slides as you want. There is no limit for slides in Solace.
